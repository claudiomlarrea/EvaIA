# EvaIA - Prototipo de Inmunología

Prototipo inicial de **EvaIA**, una plataforma de aprendizaje basado en problemas con evaluación automática orientativa para la asignatura **Inmunología** de la carrera de **Medicina**.

## Características de esta versión

- Interfaz en Streamlit
- Estilo visual institucional
- 1 caso clínico piloto
- 5 preguntas abiertas
- Evaluación automática simple por palabras clave
- Retroalimentación inmediata
- Panel docente
- Descarga de resultados en CSV y Excel
